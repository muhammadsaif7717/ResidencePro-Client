## üéØ Color Usage Guidelines

### Navigation & Headers
- **Dark:** Primary (#3B82F6) on Dark Background (#0F172A)
- **Light:** Primary (#2563EB) on White Background (#FFFFFF)

### Property Cards
- **Dark:** Card Background (#1E293B) with Slate 700 Border (#334155)
- **Light:** White Card (#FFFFFF) with Slate 200 Border (#E2E8F0)

### Buttons
- **Primary CTA:** Blue gradient (Primary ‚Üí Secondary)
- **Secondary:** Outlined with border color
- **Danger:** Error color for delete/cancel actions

### Interactive Elements
- **Hover States:** Lighten/darken by one shade
- **Active States:** Darken by two shades
- **Focus Rings:** Primary color at 50% opacity

### Stripe Payment Integration
- **Success State:** Success color (#10B981 / #059669)
- **Processing:** Info color (#06B6D4 / #0891B2)
- **Error:** Error color (#EF4444 / #DC2626)

### Map Markers (Leaflet)
- **Available Properties:** Success color
- **Pending:** Warning color
- **Sold/Unavailable:** Muted text color
- **Selected:** Primary color with pulse animation

## üí° Implementation Tips

### Tailwind Configuration
Add custom colors to `tailwind.config.js`:

```javascript
module.exports = {
  darkMode: 'class',
  theme: {
    extend: {
      colors: {
        primary: {
          DEFAULT: '#3B82F6',
          dark: '#2563EB',
        },
        secondary: {
          DEFAULT: '#8B5CF6',
          dark: '#7C3AED',
        },
      },
    },
  },
}
```

### Dark Mode Toggle
Use React state with localStorage persistence for theme switching.
